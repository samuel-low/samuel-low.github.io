Technical Documentation

I've made several technical documentations, in a varitey of technical documentation formats:

I made a User Manual for the open-source program Open Visual Trace Route. This manual details the installation requirements and methods, functions, and detailed instructions on how to use the program. This is all supplemented with figures and tables, which can be navigated through the Table of Contents. I followed a Style Guide to ensure that it would meet a certain standard.

I also documented a room's layout. This layout is network, computer and physical. The intention of this documentation is to record the layout of the room as it is. The record should accurately describe the room in enough detail that it can be recreated if desired. It has all the necessary features of Technical Documentation, A Table of Contents, Introduction, the body of information, and an Appendix.
